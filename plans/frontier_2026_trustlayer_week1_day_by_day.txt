DAY 2 (07 Apr) - API FOUNDATION
--------------------------------------------------

Objectives:
- Postaviti API skeleton i health endpoint.

Tasks:
1. U `apps/api` postaviti Fastify/NestJS (izabrati jedan, ne oba).
2. Dodati bazne rute:
   - GET /health
   - GET /v1/score/:mint (za sada mock odgovor)
3. Dodati input validaciju (Zod):
   - provera Solana mint formata.
4. Dodati centralizovan error format:
   - code
   - message
   - requestId
5. Dodati osnovni logging middleware.

Definition of done:
- `/health` radi.
- `/v1/score/:mint` vraca mock JSON po stabilnom contract-u.

--------------------------------------------------
DAY 3 (08 Apr) - SOLANA DATA INTEGRATION V1
--------------------------------------------------

Objectives:
- Povuci prve realne signale za token.

Tasks:
1. Integrisati Solana RPC provider (npr. Helius endpoint).
2. Implementirati data fetch sloj za:
   - token metadata basics
   - mint/freeze authority info
3. Implementirati holder concentration signal (top holders %).
4. Dodati timeout/retry mehanizam za external pozive.
5. Uvesti cache (Redis ili in-memory za start).

Definition of done:
- Za validan mint endpoint vraca realne signal podatke (jos bez final score formule).

3. Dodati unit testove za scoring:
   - high risk scenario

   - low risk scenario
4. Vratiti response format:
   - score
   - status (green/yellow/red)
   - reasons[]
   - timestamp

Definition of done:
- `/v1/score/:mint` vraca determinizovan score + reasons.
- Unit testovi prolaze.
